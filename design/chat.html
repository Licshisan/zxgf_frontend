lightbulb 反向传播算法 (Backpropagation) 概念解析

反向传播是训练人工神经网络的核心算法。它的主要目的是计算损失函数（Loss Function）对网络中每个权重的偏导数（梯度），从而使优化算法（如梯度下降）能够更新这些权重，最小化预测误差。简而言之，它是一个“误差向后传递，分配责任”的过程。

functions 核心数学推导 (链式法则)

假设一个简单的线性层 $z = wx + b$，激活函数为 $a = \sigma(z)$，损失函数为 $L$。

∂L / ∂w = (∂L / ∂a) · (∂a / ∂z) · (∂z / ∂w)
  1. 前向传播 (Forward Pass)： 计算每层的输出 $z$ 和 $a$，直到得出最终预测并计算损失 $L$。
  2. 计算误差项 (Compute Error)： 计算输出层的误差梯度。
  3. 反向传播误差 (Backpropagate Error)： 运用链式法则，将误差逐层向前倒推。
  4. 更新权重 (Update Weights)： $w_{new} = w_{old} - \eta \cdot (\partial L / \partial w)$，其中 $\eta$ 是学习率。
code Python (NumPy) 代码简例

import numpy as np

def sigmoid_derivative(x):
    return x * (1 - x)

# 假设我们在输出层，a2 是预测值，y 是真实值
error = y - a2
# 计算输出层 delta
d_a2 = error * sigmoid_derivative(a2)

# 计算隐藏层误差 (假设 weights2 是连接隐藏层和输出层的权重)

warning 易错点警示
  • 梯度消失： 在深层网络中使用 Sigmoid 时容易发生，导致浅层权重无法更新。
  • 维度不匹配： 矩阵乘法（Dot Product）时未正确转置矩阵。
